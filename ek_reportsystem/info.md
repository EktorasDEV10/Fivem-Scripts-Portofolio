<!-- COMMANDS -->
/togglereports  <!-- Toggle reports => /togglereports on or /togglereports off -->
/togglestreamer <!-- Toggle reports/staffchat => /togglestreamer on or /togglestreamer off -->
/staffchat <!-- Toggle Staff chat => /staffchat on or /togglereports off -->
/sm <!-- Staff chat => /sm this is a test message for other staffs -->
/report <!-- Report command => /report test report reason -->
/r <!-- Reply to a player (example => /r [id] [message] || /r 1 this is a test reply) -->
/rp <!-- Open report menu (for staff only) -->
/kick <!-- Kick command (for staff only) (example => /kick [id] [reason] || /r 1 this is a test kick) -->

<!-- FEATURES -->
Commands mentioned above are the default. You can change them in **ek_config.lua**
Messages can be also edited
Add your own reasons for report menu
Nice ui, goto and complete buttons through report menu
